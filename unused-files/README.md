# Unused files
This Node script allows finding all unused files on a Wikia wiki without having
to wait for Special:UnusedFiles' cache to update, given a list of pages in a
`list.txt file`. The wiki is hardcoded to
[thebleachfanon.wikia.com](https://thebleachfanon.wikia.com) because I was
helping with merging it with
[bleachfanfiction.wikia.com](https://bleachfanfiction.wikia.com).
